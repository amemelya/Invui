## Color Palette

### Primary Colors
- **Primary Blue**: `#0066CC` - Main CTA, headers, important elements
- **Light Blue**: `#E6F4FE` - Backgrounds, info boxes
- **Dark Blue**: `#004499` - Hover/active states

### Secondary Colors
- **Light Gray**: `#E8E8E8` - Secondary buttons, dividers, borders
- **Medium Gray**: `#CCCCCC` - Icons, placeholders
- **Dark Gray**: `#666666` - Secondary text, descriptions

### Accent Colors
- **Danger Red**: `#FF3B30` - Cancel button, logout, destructive actions
- **Success Green**: `#34C759` - Success messages (if used)

### Backgrounds
- **Page Background**: `#F5F5F5` - Main page background
- **Card Background**: `#FFFFFF` - Card/form backgrounds
- **Text Primary**: `#000000` - Main text
- **Text Secondary**: `#666666` - Supporting text

## Typography

### Font Sizes
- **Page Title**: 24px, Bold (700)
- **Section Subtitle**: 14px, Normal (400)
- **List Item Title**: 16px, Semi-bold (600)
- **List Item Description**: 14px, Normal (400)
- **Button Text**: 16px, Semi-bold (600)
- **Label Text**: 14px, Semi-bold (600)
- **Body Text**: 14px, Normal (400)
- **Small Text**: 12px, Normal (400)

## Spacing Guidelines

### Padding
- **Screen Padding**: 16px (horizontal)
- **Component Padding**: 12px (vertical)
- **Card Padding**: 16px (all sides)
- **Form Field Padding**: 10px (vertical)

### Margins
- **Section Margin**: 12px (vertical between sections)
- **Component Margin**: 8px (vertical between components)
- **List Item Spacing**: No margin (continuous)
- **Button Spacing**: 8px (between buttons)

### Border Radius
- **Cards/Inputs**: 8px
- **Buttons**: 8px
- **Modals**: 16px (top corners)

- **Mobile**: < 768px (default)

## Dark Mode Consideration

The app supports system dark mode preference via:
- `useColorScheme()` hook
- `DarkTheme` and `DefaultTheme` from React Navigation
- Currently uses light theme colors (can be expanded)

To implement full dark mode:
1. Update color palette for dark backgrounds
2. Adjust text colors for contrast
3. Update component styles with theme-aware colors
